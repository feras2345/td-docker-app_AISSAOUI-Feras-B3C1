# TD Docker App

Projet TP Docker avec API FastAPI, front en JS statique, base PostgreSQL, docker-compose, sécurité, et automatisation du déploiement.

## Structure

- api/: FastAPI app
- front/: Front statique
- db/: Scripts base
- deploy.sh: Script build et deploy
- docker-compose.yml
- .dockerignore
- .env.example
- README.md (ce fichier)
